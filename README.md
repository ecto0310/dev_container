# Dev container

My dev container

## Usage

1. Install [Dokcer](https://www.docker.com/) and [VSCode](https://code.visualstudio.com/)
1. Install [Dev Containers extension](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers) for VSCode.
1. Open the folder you want to use and select "Dev Containers: Open Folder in Container." from the command palette.
